# FinePrint

Most of us have gone through this, while applying for jobs, while buy new phones, or signing up for a new website, skipping through the entire privacy policy. Its too long and whos gonna read it. 

This app solves that. It will read the privacy policy, summarize it in a few words and highlight the pitfalls or good things that you need to know.
